# 🎮 파이썬 콘솔 게임 투표 프로그램

파이썬이랑 MariaDB를 연결해서 만든 간단한 설문조사 프로그램입니다. 
클라우드 인프라와 데이터베이스 통신을 공부하면서 실습용으로 만들어봤습니다!

## ✨ 어떤 기능이 있나요?
1. **설문 참여하기:** 목록에 있는 게임이나 내가 좋아하는 게임을 직접 입력해서 투표할 수 있습니다.
2. **현황 확인하기:** 지금까지 사람들이 투표한 결과를 모아서 총 투표수와 %를 계산해서 보여줍니다.
3. **DB 자동 연동:** 투표한 내용이 바로바로 데이터베이스에 저장됩니다.

## 🛠️ 사용한 기술
* Python 3
* MariaDB
* `pymysql` 라이브러리

## 🚀 실행하는 방법

**1. 준비물 설치**
파이썬에서 DB랑 연결하려면 아래 명령어를 터미널에 쳐서 라이브러리를 깔아야 해요.
> pip install pymysql

**2. 내 DB 정보로 수정하기**
파이썬 코드(`main.py`)를 열어서 위쪽에 있는 접속 정보를 내 환경에 맞게 바꿔주세요.
```python
conn = pymysql.connect(
    host='여기에 DB 서버 IP 주소 입력',
    user='계정 아이디',
    password='비밀번호',
    database='testdb'
)
